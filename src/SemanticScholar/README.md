cmd切换到semantic文件夹， 执行scrapy crawl txsr命令，该爬虫可自动爬取
该爬虫主要功能为有一个文献网址列表，首先爬取第一个文章的参考文献和被引文献，加入文献网址列表，然后依次爬取列表中的文献，同时爬取时也将其参考文献和被引文献加入文献网址列表，并且进行去重操作，保证文献没有重复。
txsr第22行为第一个文章的地址，改成自己想要的即可。
